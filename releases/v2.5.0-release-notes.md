## 部署说明

### 前置条件
- Node.js >= 18.0.0
- Docker >= 20.10.0
- Docker Compose >= 2.0.0

### 快速部署
```bash
# 拉取最新代码
git pull origin main

# 构建并运行
docker-compose up -d --build

# 查看日志
docker-compose logs -f app
```

### CI/CD 自动部署
推送代码到 main 分支后，GitHub Actions 将自动触发：
1. 代码检查（ESLint + TypeScript）
2. 单元测试（Vitest）
3. 构建产物（Vite）
4. Docker 镜像构建
5. 部署到生产环境
6. 健康检查

## 升级建议

### 从 v2.4.x 升级
```bash
# 1. 备份数据库
mysqldump -u root -p ai_work > backup.sql

# 2. 拉取最新代码
git pull origin main

# 3. 安装依赖
npm ci

# 4. 运行数据库迁移
npm run migrate

# 5. 重建前端
npm run build

# 6. 重启服务
pm2 restart all
```
